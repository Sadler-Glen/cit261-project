Topic 09: Standard JavaScript Events Including those for Mobile Devices ( Ex. onTouchBegin, onLoad, etc.) and Animation and Transition Events.

Various events

Example 1

The following is an example of an onload event. The changeColor() function which changes the color from sky blue to red is called when this page is loaded. There are also examples of onclick onmouseover, and onmouseout, events. When the user places his mouse over(onmouseover) the 'myBox'<div> the box fades out. When the user moves the mouse away(onmouseout) the <div> fades in. I have also incorporated an ontouchstart and an ontouchend pair to fade-out and fade-in the <div> on touch devices. The event handlers are, in these cases, Javascript functions which are called when these events occur.

Example 2

The following is an example of onkeyup and onblur. When the user clicks inside the input field for either the First Name or Last Name input fields and then leaves either field empty the validation rule causes the field color to turn red. By the same token, if text is typed into either field, and as soon as the key is released after the the first key-press, the onkeyup validation rule causes the field to turn green. The onblur calls the validate function when the input fields lose focus. Try it out!
