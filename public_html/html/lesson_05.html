Topic 05: Local Storage API, Storing and Retrieving Simple Data, Arrays, Associative Arrays, and Objects

Local Storage & Session Storage

HTML5 Local Storage & Session Storage are a part of the Web storage API and these two objects give a way that web pages can be stored as named key/value pairs within a browser. This is similar to cookies, where the data exists even when the browser tab is closed, or when you browse to another site, or when you exit the browser. Local storage data will still be there in a week, a month, or a years time, however Session storage disappears when the browser tab is closed.

  • Depending on device, local storage can hold 2 -100Mb+ of data
  • Saved as string format such as JSON
  • Useful for web interface data
  • Helpful for temporary or offline data
  • Can read, write, and delete data
  • Must parse when reading
  • Must stringify data when writing

Local Storage object and methods

localStorage.setItem() - stores data with no expiration date
localStorage.getItem() - retrieves from local storage
localStorage.removeItem() - removes from local storage
localStorage.clear() - clears all storage
sessionStorage.setItem() - stores data for one session (data is lost when the browser tab is closed)
sessionStorage.getItem() - retrieves from local storage
sessionStorage.removeItem() - removes from local storage
