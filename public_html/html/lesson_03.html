Topic 03: JSON Parse, Stringify

Definition

JSON is an acronym for Javascript Object Notation. It is a standard that uses text to represent Javascript objects. It is used in server-client data transmission in web applications eg., sending data from server to client so it can be displayed in a browser, or vice versa.

There are a lot of really cool sites out there where you can generate JSON objects, for testing purposes, which should not to be confused with Javascript objects. One of those sites is filltext.com where you can create JSON objects. There is also another site called JSONlint where you can validate your JSON objects.

JSON Structure

A JSON object is a string which has similar appearance to Javascript object literal. You are able to include the same basic data types, like strings, numbers, booleans, etc. See below an example of a JSON object which I created at www.filltext.com in a matter of seconds, once you get to know how to do it. For those interested to create this I typed the following to produce the JSON object. Really Cool!

http://www.filltext.com/?rows=3&pretty=true&ID={index}&fname={firstName}&lname={lastName} &business={business}&address={addressObject}
