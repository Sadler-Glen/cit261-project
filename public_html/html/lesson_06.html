Topic 06: DOM Manipulation Using createElement, appendChild, insertBefore, removeChild, etc.

What is the DOM?

The Document Object Model (DOM) is the interface for HTML documents and is the page that programs are able to change the document structure, style and content. The DOM shows the document as nodes and objects and in this way the programming language such as Javascript can connect to the page. The web page is such a document which can be either displayed as HTML source code or within the browser window. Both of these are the same document. The DOM is an object-oriented representation of a web page, that can be modified with Javascript.
