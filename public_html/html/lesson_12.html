Transform Example 1

This rotate transformation is setup with a 2.5 second delay to start. Also included in this is a transition of color from magenta to blue. Their is also a transition of the box size and text size. This transform is triggered in CSS by the setInterval(). The rotation is set to 270°.

Transform Example 2

This example uses transform: rotate and also transition: 300px to accomplish the task of moving(transform) from one place to another, and transform: rotate) to make the circle appear as a wheel from one position to another. The rotate is set to 720°. The method to activate the animation is the #wheel:hover in our CSS.
