Topic 07: Manipulating CSS Class Properties Using JavaScript.

Overview

When it comes to Javascript we do not want to manipulate the CSS directly. It is best to have CSS set up that can do the manipulation. This is helpful when collaborating as a team. One of the team could be working on the CSS and another on the Javascript and they do not have to worry about each other. Ideally our Javascript should only be selecting classes and mostly only be manipulating classes.

The two ways to manipulate classes in plain Javascript are with className and classList. Using className is a more direct approach and will work in every browser, whereas using classList is much simpler but may not be supported in every browser. We will focus our efforts here on classList

The following are the methods of the classList API.

  • add(String [,String])
  • remove(String [,String])
  • item(Number)
  • toggle(String [,force])
  • contains(String)
  • replace(oldClass, newClass)
